# 💬 Quantum Chat App

A modern cross-platform chat application built using Flutter.  
It supports real-time messaging, secure authentication, and a clean UI across Android, iOS, Web, and Desktop platforms.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 💬 Real-time chat messaging
- 👥 Contact management system
- 🔒 Secure data handling
- 🌙 Light & Dark mode support
- 📱 Fully responsive UI
- 🌐 Cross-platform (Android, iOS, Web, Windows, macOS, Linux)
- ⚡ Fast and smooth performance

---

## 🛠️ Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** Firebase / Custom backend (if used)
- **Database:** Firestore / Realtime Database
- **Authentication:** Firebase Auth
- **State Management:** Provider
- **Architecture:** MVC / Clean Architecture
